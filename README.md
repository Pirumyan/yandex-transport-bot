# Telegram Transport Bot

Бот для получения скриншотов с Яндекс.Карт для конкретных остановок (Комитас и Сарян).

## Как настроить (Локально)
1. Установите Python 3.11
2. Создайте виртуальное окружение: `python3 -m venv venv`
3. Активируйте его: `source venv/bin/activate` (для Mac)
4. Установите зависимости: `pip install -r requirements.txt`
5. Установите браузер: `playwright install chromium`
6. Запустите: `BOT_TOKEN="ваш_токен_здесь" python main.py`

## Настройка ссылок (Важно ⚠️)
В файле `main.py` есть словарь `STOPS`. Вам нужно зайти на Яндекс.Карты, найти нужную остановку, включить слой "Движущийся транспорт" (автобусы) и скопировать ссылку из адресной строки. 
Замените заглушки в коде на ваши реальные ссылки, чтобы бот скринил именно тот участок карты, который вам нужен.

## Как запустить бесплатно и 24/7
Поскольку боту нужен реальный браузер (Playwright) для скриншотов, бесплатные сервисы на 512 ОЗУ (типа Render) могут не всегда справляться, плюс Render отключает бесплатные сервисы, если нет входящих HTTP-запросов.

Лучший вариант для бесперебойной работы 24/7 абсолютно бесплатно — **Oracle Cloud (Always Free)**.
Они дают сервер с характеристиками (до 24 ГБ ОЗУ, 4 ядра ARM), чего хватит с головой. 

Шаги:
1. Регистрируетесь в Oracle Cloud и создаете бесплатный инстанс (Compute Instance) на базе Ubuntu (Ampere ARM).
2. Подключаетесь к серверу через терминал по SSH.
3. Устанавливаете Docker на сервер: `sudo apt install docker.io -y`
4. Переносите эти файлы (`main.py`, `requirements.txt`, `Dockerfile`) на сервер.
5. Собираете образ: `sudo docker build -t transport_bot .`
6. Запускаете 24/7 в фоне: `sudo docker run -d --restart unless-stopped --name transport_bot -e BOT_TOKEN="ВАШ_ТОКЕН_ИЗ_BOTFATHER" transport_bot`

Все! Бот будет работать круглосуточно.
